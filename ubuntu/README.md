# Vagrant

## Usage
```
#
# If you have never create instance before
#

$ vagrant init ubuntu/xenial64
$ vagrant up --provider virtualbox
$ vagrant ssh

#
# Check if instance running
#
$ vagrant status

#
# Reload 
#
$ vagrant reload --provision

#
# Plugin list
#
$ vagrant plugin list

```
